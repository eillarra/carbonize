========================
Welcome to ``carbonize``
========================

``carbonize`` is a collection of carbon footprint calculators. It provides a simple and consistent API
for calculating carbon emissions for various activities and processes. Whether you're building
a sustainability dashboard, conducting environmental research, or just curious about your carbon
footprint, ``carbonize`` offers tools to help quantify environmental impact.

The library aims to make carbon calculations accessible while maintaining scientific accuracy
based on established methodologies and emission factors.
